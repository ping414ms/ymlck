# ymlck
Simple YAML syntax checker made with Bash script.

## Usage

```
ymlck [-p] [file] 
```

or with using pipe

```
cat somefile | ymlck
```

### Install

Copy bin/ymlck to some executable directory, and set executable permission to that.

### Options

#### -p 
Print JSON text that converted from loaded YAML file.
